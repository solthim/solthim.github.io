# 솔팀 (Solthim) — 사업자 인건비 공유 앱 데모

회계법인이 발행한 인건비 자료를 사업자(사장님)가 휴대전화로 조회·입력하는 웹앱의 **데모**입니다.

- **데모 PIN: `12345`**
- 데이터는 전부 가짜이며(`demo-data.js`), 서버 없이 브라우저 안에서만 동작합니다.
- "담당자에게 제출"을 눌러도 아무것도 전송·저장되지 않습니다.

## 기능

- 인건비 요약: 월 합계, 직원 구성(월급제·시급제·사업소득), 직원별 급여·수당·보험유형
- 직원 탭 → 근무 입력: 달력으로 근무시간 입력(복수 선택 일괄 적용) 또는 총 시간 입력
- 수당 추가, 월중 입사·퇴사 시 월급 일할계산 자동 미리보기
- 직원 추가(이름 + 급여유형), 주민번호 패턴 입력 차단

## 파일

| 파일 | 내용 |
|---|---|
| `index.html` | 앱 본체 — 원본은 내부 프로젝트의 `web/index.html`, `demo-data.js` 로드 한 줄만 추가됨 |
| `demo-data.js` | 가짜 발행 스냅샷 (`window.SOLTHIM_DEMO`) |

실제 서비스는 토큰 링크 + PIN 인증, 발행형 스냅샷, 검토 후 반영 구조로 동작합니다.
